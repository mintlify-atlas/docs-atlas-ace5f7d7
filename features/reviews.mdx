## Overview

ArrendaOco's review system allows tenants to rate and review properties they've experienced, helping future renters make informed decisions.

## Review Model

### Database Schema

From app/Http/Controllers/ResenaController.php:28-57:

```php
public function store(Request $request, Inmueble $inmueble)
{
    $request->validate([
        'puntuacion' => 'required|integer|min:1|max:5',
        'comentario' => 'required|string|max:1000',
    ]);
    
    // Prevent landlords from reviewing their own properties
    if ($inmueble->propietario_id === Auth::id()) {
        return back()->with('error', 'No puedes calificar tu propia propiedad.');
    }
    
    // Prevent duplicate reviews
    $resenaExistente = Resena::where('usuario_id', Auth::id())
                            ->where('inmueble_id', $inmueble->id)
                            ->first();
    
    if ($resenaExistente) {
        return back()->with('error', 'Ya has calificado esta propiedad.');
    }
    
    Resena::create([
        'usuario_id' => Auth::id(),
        'inmueble_id' => $inmueble->id,
        'puntuacion' => $request->puntuacion,
        'comentario' => $request->comentario,
    ]);
    
    return back()->with('success', 'Tu reseña ha sido enviada con éxito.');
}
```

<Warning>
**Business Rules:**
- Users cannot review their own properties
- Users can only leave **one review per property**
- Reviews are permanent (users can update but not delete their own reviews without admin)
</Warning>

## Updating a Review

**Web Route**: `PUT /resenas/{resena}`

**API Route**: `PUT /api/resenas/{resena}`

**Authorization**: Only the review author

**Implementation** (app/Http/Controllers/ResenaController.php:62-79):

```php
public function update(Request $request, Resena $resena)
{
    if ($resena->usuario_id !== Auth::id()) {
        abort(403, 'No tienes permiso para editar esta reseña.');
    }
    
    $request->validate([
        'puntuacion' => 'required|integer|min:1|max:5',
        'comentario' => 'required|string|max:1000',
    ]);
    
    $resena->update([
        'puntuacion' => $request->puntuacion,
        'comentario' => $request->comentario,
    ]);
    
    return back()->with('success', 'Reseña actualizada correctamente.');
}
```

## Deleting a Review

**Web Route**: `DELETE /resenas/{resena}`

**API Route**: `DELETE /api/resenas/{resena}`

**Authorization**: Review author or admin

**Implementation** (app/Http/Controllers/ResenaController.php:84-95):

```php
public function destroy(Resena $resena)
{
    $user = Auth::user();
    
    // Author or admin can delete
    if ($resena->usuario_id === $user->id || 
        $user->es_admin || 
        $user->tieneRol('admin')) {
        $resena->delete();
        return back()->with('success', 'Reseña eliminada correctamente.');
    }
    
    abort(403, 'No tienes permiso para eliminar esta reseña.');
}
```

## Admin Moderation

Admins can view all reviews for moderation purposes.

**Route**: `GET /admin/resenas`

**Access**: Admin only

**Implementation** (app/Http/Controllers/ResenaController.php:15-23):

```php
public function index()
{
    if (!Auth::user()->es_admin && !Auth::user()->tieneRol('admin')) {
        abort(403);
    }
    
    $resenas = Resena::with(['usuario', 'inmueble'])
        ->latest()
        ->paginate(20);
    
    return view('admin.resenas.index', compact('resenas'));
}
```

**Features**:
- View all reviews system-wide
- Delete inappropriate reviews
- See which users are leaving reviews
- Monitor property ratings

## Best Practices

### For Users

1. **Be honest**: Share your genuine experience
2. **Be specific**: Mention what you liked and didn't like
3. **Be respectful**: Avoid offensive language
4. **Update if needed**: You can edit your review later

### For Landlords

1. **Respond to reviews**: Engage with feedback (feature can be added)
2. **Improve based on feedback**: Use reviews to enhance properties
3. **Don't review your own properties**: System prevents this

### For Admins

1. **Monitor for spam**: Check new reviews regularly
2. **Remove inappropriate content**: Use delete function for offensive reviews
3. **Ensure fairness**: Don't delete negative but honest reviews
