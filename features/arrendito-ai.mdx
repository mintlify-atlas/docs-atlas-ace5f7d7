## Overview

Arrendito (also known as ROCO) is ArrendaOco's AI-powered chat assistant that helps users find properties, answer questions about rentals, and provide personalized recommendations using Google's Gemini 2.5 Flash model.

## How It Works

Arrendito combines:

1. **Intelligent Property Search**: Analyzes user messages to find relevant properties
2. **Google Gemini AI**: Generates natural, helpful responses
3. **Local Context**: Understands Ocosingo geography (UTS, Centro, etc.)
4. **Personality**: Responds as a friendly Beagle assistant

## Implementation

From app/Http/Controllers/ArrenditoChatController.php:24-128:

```php
public function chat(Request $request)
{
    $request->validate(['message' => 'required|string|max:500']);
    $userMessage = $request->message;
    $apiKey = config('services.gemini.api_key', env('GEMINI_API_KEY'));
    
    if (empty($apiKey)) {
        return response()->json([
            'success' => false,
            'response' => '¡Guau! 🐾 Mi cerebro perruno necesita una llave API para funcionar.'
        ]);
    }
    
    // Search for relevant properties
    $inmuebles = $this->buscarInmueblesInteligente($userMessage);
    
    // Build context with found properties
    $contexto = $this->construirContextoInmuebles($inmuebles);
    
    // Get system instruction (personality)
    $systemInstruction = $this->getSystemInstruction();
    
    // Combine user message with property context
    $userContent = "INMUEBLES QUE 'OLFATEASTE' PARA ESTA PREGUNTA:\n{$contexto}\n\nUsuario dice: {$userMessage}";
    
    try {
        $url = "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key={$apiKey}";
        
        $response = Http::withoutVerifying()
            ->timeout(30)
            ->post($url, [
                'systemInstruction' => [
                    'parts' => [['text' => $systemInstruction]]
                ],
                'contents' => [
                    ['role' => 'user', 'parts' => [['text' => $userContent]]]
                ],
                'generationConfig' => [
                    'temperature' => 0.8,
                    'maxOutputTokens' => 1024,
                ],
            ]);
        
        if ($response->successful()) {
            $data = $response->json();
            $resText = $data['candidates'][0]['content']['parts'][0]['text'];
            return response()->json(['success' => true, 'response' => $resText]);
        }
        
        return response()->json([
            'success' => false,
            'response' => '¡Guau! 🐾 Tuve un problemita. ¿Podrías intentar de nuevo?'
        ]);
        
    } catch (\Exception $e) {
        Log::error("ROCO Chat Error: " . $e->getMessage());
        return response()->json([
            'success' => false,
            'response' => '¡Guau! 🐾 Error inesperado. Contacta a arrendaoco@gmail.com'
        ]);
    }
}
```

## Context Building

Arrendito formats property information for Gemini.

From app/Http/Controllers/ArrenditoChatController.php:216-230:

```php
private function construirContextoInmuebles($inmuebles): string
{
    if ($inmuebles->isEmpty()) {
        return "No se encontraron inmuebles disponibles actualmente.";
    }
    
    $contexto = "";
    foreach ($inmuebles as $i) {
        $url = route('inmuebles.show', $i->id);
        $tipo = $i->tipo ? " ({$i->tipo})" : "";
        $contexto .= "🏠 <b>{$i->titulo}</b>{$tipo}<br>💰 \${$i->renta_mensual}<br>📍 {$i->direccion}<br>🔗 BOTÓN_URL: {$url}<br><br>";
    }
    
    return $contexto;
}
```

    // Clear input
    document.getElementById('user-message').value = '';
});

## Performance Considerations

- **Timeout**: 30 seconds for Gemini API calls
- **Rate Limiting**: Gemini Free tier has quotas (429 errors)
- **Caching**: Consider caching common questions
- **Property Limit**: Returns max 5 properties per search

## Privacy & Security

- **No authentication required**: Public chat for all visitors
- **Message length**: Limited to 500 characters
- **No conversation history**: Each message is stateless
- **SSL verification disabled**: For development (`withoutVerifying()`)

<Tip>
In production, remove `withoutVerifying()` and enable SSL certificate verification.
</Tip>
