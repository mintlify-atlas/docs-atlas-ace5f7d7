<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with ArrendaOco in minutes</p>
  
  <Steps>
    <Step title="Install and Configure">
      Clone the repository and install dependencies. ArrendaOco is built with Laravel 12 and Vue 3.
      
      ```bash
      git clone https://github.com/edagrex-sketch/Arrendaoco.git
      cd Arrendaoco
      composer install
      npm install
      ```
      
      Copy the environment file and generate your application key:
      
      ```bash
      cp .env.example .env
      php artisan key:generate
      ```
    </Step>
    
    <Step title="Set Up the Database">
      Configure your database connection in `.env` and run migrations:
      
      ```bash
      php artisan migrate
      ```
      
      <Note>
        ArrendaOco supports MySQL, PostgreSQL, and SQLite. See the [Database Configuration](/configuration/database) guide for details.
      </Note>
    </Step>
    
    <Step title="Start the Development Server">
      Run the development servers for both backend and frontend:
      
      ```bash
      php artisan serve
      npm run dev
      ```
      
      Your application will be available at `http://localhost:8000`
    </Step>
    
    <Step title="Create Your First User">
      Register a new user account through the web interface or use the API:
      
      ```bash
      curl -X POST http://localhost:8000/api/register \
        -H "Content-Type: application/json" \
        -d '{
          "nombre": "John Doe",
          "email": "john@example.com",
          "password": "securepassword",
          "password_confirmation": "securepassword"
        }'
      ```
      
      <Accordion title="Example Response">
        ```json
        {
          "user": {
            "id": 1,
            "nombre": "John Doe",
            "email": "john@example.com",
            "estatus": "activo",
            "roles": ["inquilino"]
          },
          "token": "1|laravel_sanctum_token_here"
        }
        ```
      </Accordion>
    </Step>
  </Steps>
</div>
