## Overview

Chat with ROCO, ArrendaOco's intelligent Beagle assistant! ROCO is powered by Google Gemini 2.5 Flash and provides context-aware responses about properties, rental questions, and location-specific information for Ocosingo, Chiapas.

<Note>
  **No Authentication Required**
  
  This endpoint is publicly accessible and does not require authentication, making it perfect for guest users exploring the platform.
</Note>

## Key Features

- **Intelligent Property Search**: ROCO automatically searches for relevant properties based on your message
- **Location Awareness**: Knows Ocosingo geography (UTS, Centro, zones)
- **Price Detection**: Understands budget queries and price ranges
- **Type Recognition**: Detects property types (casa, departamento, cuarto, local, terreno)
- **Conversational AI**: Friendly, dog-themed personality
- **HTML Responses**: Returns formatted HTML with buttons and links

## Request

### Body Parameters

<ParamField body="message" type="string" required>
  The user's message to ROCO (max 500 characters)
  
  **Examples:**
  - "Busco un departamento cerca de la UTS por menos de $4000"
  - "¿Qué casas baratas tienes?"
  - "Necesito un cuarto en el centro"
  - "¿Cómo funciona el proceso de renta?"
</ParamField>

### Context Awareness

ROCO understands Ocosingo's key locations:

- **UTS (Universidad Tecnológica de la Selva)**: Located on the outskirts near the Altamirano highway
- **Centro**: Downtown area with the park and market
- **Zona Norte**: Northern neighborhoods near the Palenque highway
- **Zona Sur**: Southern area toward San Cristóbal highway

## Intelligent Property Search

ROCO automatically searches properties based on message content:

### Location Detection

<CodeGroup>
```php UTS/University
if (str_contains($mensaje, 'uts') || str_contains($mensaje, 'universidad')) {
    $query->where(function ($q) {
        $q->where('direccion', 'LIKE', '%UTS%')
```

### Price Detection

<CodeGroup>
```php Maximum Price
// Detects: "menos de $4000", "$3500", "3000 pesos"
if (preg_match('/menos\s*de\s*\$?(\d[\d,]*)/i', $mensaje, $matches)) {
    $precio = (int) str_replace(',', '', $matches[1]);
    $query->where('renta_mensual', '<=', $precio);
}
```

```php Budget-Friendly
// Detects: "barato", "económico"

}
```
</CodeGroup>

## Rate Limiting

<Note>
  **Timeout**: 30 seconds per request
  
  If Gemini doesn't respond within 30 seconds, the request will timeout and return a connection error message.
</Note>

Google Gemini API has its own rate limits (429 errors). ROCO handles these gracefully with friendly error messages.

## Error Handling

All errors are logged server-side and return user-friendly messages:

```php
Log::error("ROCO Chat - Gemini API Error [{$statusCode}]: {$errorMessage}", [
    'model' => self::GEMINI_MODEL,
    'status' => $statusCode,
    'body' => $response->body(),
]);
```
