---

## Overview

This endpoint returns the complete profile of the currently authenticated user, including their roles and permissions.

## Authentication

<Note>
  This endpoint requires authentication. Include your Bearer token in the Authorization header.
</Note>

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer {token}
</ParamField>

## Response

<ResponseField name="id" type="integer">
  User ID
</ResponseField>

<ResponseField name="nombre" type="string">
  Full name
</ResponseField>

<ResponseField name="email" type="string">
  Email address
</ResponseField>

<ResponseField name="foto_perfil" type="string">
  Profile photo URL (null if not set)
</ResponseField>

<ResponseField name="estatus" type="string">
  Account status: `activo` or `inactivo`
</ResponseField>

<ResponseField name="es_admin" type="boolean">
  Whether user has admin privileges
</ResponseField>

<ResponseField name="roles" type="array">
  Array of role names: `inquilino`, `propietario`, `admin`
</ResponseField>

<ResponseField name="created_at" type="timestamp">
  Account creation date
</ResponseField>

## Examples

### Request

```bash cURL
curl -X GET https://arrendaoco.com/api/me \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

```javascript JavaScript
const response = await fetch('https://arrendaoco.com/api/me', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN_HERE'
  }
});
const user = await response.json();
```

### Response

```json
{
  "id": 1,
  "nombre": "John Doe",
  "email": "john@example.com",
  "foto_perfil": "https://arrendaoco.com/storage/perfil/photo.jpg",
  "estatus": "activo",
  "es_admin": false,
  "roles": ["inquilino", "propietario"],
  "created_at": "2024-01-15T10:30:00.000000Z"
}
```

## Error Responses

### 401 Unauthorized

```json
{
  "message": "Unauthenticated."
}
```

- Display user information in the application header
