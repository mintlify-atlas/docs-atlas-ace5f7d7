## Overview

Returns a list of all rental contracts where the authenticated user is either the property owner (landlord) or the tenant. Each contract includes property details, participant information, dates, and financial terms.

## Implementation Details

### Query Logic

The endpoint uses an OR condition to fetch contracts:

```php
Contrato::with(['inmueble', 'propietario', 'inquilino'])
    ->where('propietario_id', $usuario->id)
    ->orWhere('inquilino_id', $usuario->id)
    ->latest()
    ->get();
```

This returns contracts where the user is:
- The property owner (`propietario_id`)
- OR the tenant (`inquilino_id`)

### Ordering

Contracts are returned in descending order by creation date (newest first).
