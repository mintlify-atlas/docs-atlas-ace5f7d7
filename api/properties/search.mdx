## Endpoint

```
GET /api/inmuebles/public-list
```

## Description

Search and filter available properties using multiple criteria. This endpoint is public and does not require authentication. Results are paginated and ordered by most recent first.

<Info>
  This endpoint is the same as the [List Properties](/api/properties/list) endpoint but emphasizes the filtering and search capabilities.
</Info>

## Query Parameters

### Filters

<ParamField query="tipo" type="string" optional>
  Filter by property type (exact match)
  
  Common values:
  - `Casa` - House
  - `Departamento` - Apartment
  - `Local` - Commercial space
  - `Terreno` - Land
  - `Oficina` - Office
</ParamField>

<ParamField query="ciudad" type="string" optional>
  Filter by city name (partial match, case-insensitive)
  
  Examples:
  - `Ocosingo`
  - `San Cristóbal`
  - `Tuxtla`
</ParamField>

### Pagination

<ParamField query="page" type="integer" default="1" optional>
  Page number for pagination
</ParamField>

<ParamField query="per_page" type="integer" default="15" optional>
  Number of results per page (max: 100)
</ParamField>

## Search Examples

### Search by Property Type

Find all houses (`Casa`):

```bash cURL
curl -X GET "https://api.arrendaoco.com/api/inmuebles/public-list?tipo=Casa" \
  -H "Accept: application/json"
```

### Search by City

Find properties in Ocosingo:

```bash cURL
curl -X GET "https://api.arrendaoco.com/api/inmuebles/public-list?ciudad=Ocosingo" \
  -H "Accept: application/json"
```

### Combined Filters

Find apartments in San Cristóbal:

```bash cURL
curl -X GET "https://api.arrendaoco.com/api/inmuebles/public-list?tipo=Departamento&ciudad=San%20Cristóbal" \
  -H "Accept: application/json"
```

### With Pagination

Get second page of results (10 per page):

```bash cURL
curl -X GET "https://api.arrendaoco.com/api/inmuebles/public-list?page=2&per_page=10" \
  -H "Accept: application/json"
```

## JavaScript Examples

### Basic Search Component

```javascript React
import { useState, useEffect } from 'react';

function PropertySearch() {
  const [properties, setProperties] = useState([]);
  const [filters, setFilters] = useState({
    tipo: '',
    ciudad: '',
    page: 1
  });
  const [loading, setLoading] = useState(false);

  useEffect(() => {
    async function searchProperties() {
      setLoading(true);
      
      const params = new URLSearchParams();
      if (filters.tipo) params.append('tipo', filters.tipo);
      if (filters.ciudad) params.append('ciudad', filters.ciudad);
      params.append('page', filters.page);

      const response = await fetch(
        `https://api.arrendaoco.com/api/inmuebles/public-list?${params}`,
        {
          headers: { 'Accept': 'application/json' }
        }
      );
      
      const data = await response.json();
      setProperties(data.data);
      setLoading(false);
    }

    searchProperties();
  }, [filters]);

  return (
    <div>
      <input
        type="text"
        placeholder="Ciudad"
        value={filters.ciudad}
        onChange={(e) => setFilters({ ...filters, ciudad: e.target.value })}
      />
      
      <select
        value={filters.tipo}
        onChange={(e) => setFilters({ ...filters, tipo: e.target.value })}
      >
        <option value="">Todos los tipos</option>
        <option value="Casa">Casa</option>
        <option value="Departamento">Departamento</option>
        <option value="Local">Local</option>
      </select>

      {loading ? (
        <p>Cargando...</p>
      ) : (
        <div>
          {properties.map(property => (
            <div key={property.id}>
              <h3>{property.titulo}</h3>
              <p>${property.renta_mensual}/mes</p>
            </div>
          ))}
        </div>
      )}
    </div>
  );
}
```

### Advanced Filtering Helper

```javascript Utility Function
/**
 * Build search URL with filters
 */
function buildSearchURL(baseURL, filters) {
  const params = new URLSearchParams();
  
  // Add filters
  if (filters.tipo) params.append('tipo', filters.tipo);
  if (filters.ciudad) params.append('ciudad', filters.ciudad);
  
  // Add pagination
  if (filters.page) params.append('page', filters.page);
  if (filters.per_page) params.append('per_page', filters.per_page);
  
  return `${baseURL}?${params.toString()}`;
}

// Usage
const url = buildSearchURL('https://api.arrendaoco.com/api/inmuebles/public-list', {
  tipo: 'Casa',
  ciudad: 'Ocosingo',
  page: 1,
  per_page: 20
});

const response = await fetch(url, {
  headers: { 'Accept': 'application/json' }
});
```

## Implementation Details

**Source:** `app/Http/Controllers/Api/InmuebleController.php:21`

### Query Building

```php
$query = Inmueble::with(['propietario', 'imagenes', 'resenas.usuario'])
    ->where('estatus', 'disponible');

// Type filter (exact match)
if ($request->has('tipo')) {
    $query->where('tipo', $request->tipo);
}

// City filter (partial match)
if ($request->has('ciudad')) {
    $query->where('ciudad', 'like', '%' . $request->ciudad . '%');
}

return InmuebleResource::collection(
    $query->latest()->paginate(15)
);
```

### Filter Behavior

**Type Filter (`tipo`):**
- Exact match required
- Case-sensitive
- Use exact values like "Casa", "Departamento"

**City Filter (`ciudad`):**
- Partial match supported
- Case-insensitive (SQL LIKE)
- Matches anywhere in the city name
- Example: `ciudad=Oco` matches "Ocosingo"

**Status Filter:**
- Automatically filters to only show `estatus = 'disponible'`
- Cannot be overridden via query parameters

### Sorting

- Results are always sorted by most recent first (`latest()`)
- Based on `created_at` timestamp
- Cannot be customized via query parameters

### Client-Side Filtering

For filters not supported by the API, you can filter results client-side:

```javascript
// Fetch all results
const response = await fetch('https://api.arrendaoco.com/api/inmuebles/public-list?per_page=100');
const data = await response.json();

// Filter by price range
const filtered = data.data.filter(property => 
  property.renta_mensual >= 3000 && 
  property.renta_mensual <= 8000
);

// Filter by bedrooms
const threeBedroomHomes = data.data.filter(property => 
  property.habitaciones === 3
);

// Sort by price
const sortedByPrice = [...data.data].sort((a, b) => 
  a.renta_mensual - b.renta_mensual
);
```

## Notes

- Only properties with `estatus = 'disponible'` are returned
- Results include full property details, owner info, images, and reviews
- All filters are optional - omit them to get all available properties
- The `ciudad` filter supports partial matching for better UX
- Maximum 100 results per page
- Public endpoint - no authentication required
